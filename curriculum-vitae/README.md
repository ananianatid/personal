# Curriculum Vitae - Anatide ANANI

CV en ligne, imprimable, avec le même design que le portfolio.

## Tech stack

- HTML / CSS / JavaScript vanilla
- Même système de design que `portfolio/` (polices, couleurs, thème clair/sombre)

## Démarrer

Ouvrir `index.html` dans un navigateur. Aucune étape de build nécessaire.

## Impression

Le CV est optimisé pour l'impression A4. Utiliser la fonction d'impression du navigateur (Ctrl+P / Cmd+P).
